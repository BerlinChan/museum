***********精灵岛游戏说明**************

感谢您下载《精灵岛II》游戏!
这是我继《潜艇大战》的第二款力作。
★请注意★:本游戏不要在Basic助手下运行,请在GVBasic语言(启动方式:按下文曲星[计算]按钮)中运行进入GVBasic语言环境,输入以下命令
LOAD"JLD" +[输入]
RUN +[输入]

////////////////////////////////
一、游戏操作说明:
　1.[左][右]方向键可以控制精灵左右移动
　2.[Q].[A].[Z]键或者方向键[上]可以让精灵跳跃.这几个键功能一样　　　,仅仅为了操作方便而已.
　3.在游戏中按[X]键可以跳出游戏.
　4.游戏中按[P]键暂停游戏.
　5.当精灵跳起后按方向键[下]可以让精灵不再上跳而改为紧急降落.
//////////////////////////////////
二、游戏规则:
　1.精灵每一局都必须吃10个心形宝物才能过关,过关后难度自动提高
　2.精灵每局吃的心的数量在屏幕的左下方显示,当进度跳满后[10个宝物　　]就过关
　3.精灵必须通过跳跃来跨过悬崖,如果精灵不小心掉入悬崖便为死亡.
　4.精灵从两层悬崖上下移动的时候必须乘坐屏幕左边的[电梯],!切记　　　!不可以让精灵从上层悬崖直接跳到下层悬崖,否则精灵会死亡.

///////////////////////////////////
三、游戏技巧:
　1.[安全伞]:精灵在每一局拥有三把[安全伞],它的数量显示在屏幕的左　　中部分.[安全伞]的主要作用是,可以让精灵在跳起后能够在空中再一　　次跳起.这样精灵在遇到危险的时候可以使用[安全伞](再一次按跳键　　)使自己着陆到安全的地方.
　2.精灵在跳起的时候可以使用方向键控制她在空中的移动方向,这样就　　可以控制她的着陆点了.
